# @axeth/api

Shared and versioned APIs for the **Axeth** ecosystem.

`@axeth/api` provides stable interfaces, schemas, and helpers that abstract
Minecraft Bedrock Edition versions, enabling Axeth tools to remain compatible
across game updates.

This package is designed to be used by Axeth core, plugins, and advanced tooling.
